背景

为什么写另一个单位转换工具？已经有很多了。好吧，我找不到我想要的那一种。

使用ConvertAll，您可以以任何方式组合您想要的单位。如果您想转换英寸每十年，没关系。或者是米-磅，或者是立体海里。这些单位不必对其他人有意义。

因为我不是在软件业，所以只要没有并入任何专有程序，我制作的这个软件免费提供给任何人使用、分发和修改。如果您喜欢这个软件，可以随意让其他人知道它。也可以让我知道您的想法 - 我的电子邮箱地址是doug101 AT bellz DOT org

功能

  • 可以键入（含自动完成）用于转换的基本单位或者从列表中选择。.
  • 当输入一个单位时，单位列表将自动过滤，仅显示含有匹配字词的单位。
  • 一般来说，可以使用缩写或者全称来选择单位。
  • 可以与“*”和“/”运算符组合单位。
  • 可以用“^”运算符将单位提升到幂（平方、立方、等等) 。
  • 分母中的单位可以用括号分组。
  • 具有非线性尺度的单位，比如温度，也可以转换。
  • 可以从菜单选取最近使用的单位组合
  • 可以在“源单位”或“目标单位”单位任一方输入数字，可以双向转换。
  • 可以输入基本的数学表达式代替数字。
  • 选项控制数值结果的格式。
  • 单位数据包含500多个单位。
  • 单位数据文件的格式使得添加其他单位变得容易。
  • 命令行选项可用于进行转换，无需GUI。
  • 用户界面和单位数据包含中文、英语、法语、德语、俄语和西班牙语。

系统要求

Linux

ConvertAll需要以下库：

  • Python（版本3.4或更高）
  • Qt（版本5.4或更高 - 查看Qt Project获取更多信息）
  • PyQt（版本5.4或更高 - 查看Riverbank获取更多信息）

Windows

使用二进制分发中提供的文件，ConvertAll应该可以在任何运行Windows XP、Vista、7、8或10的计算机上运行。

安装

Linux

从convertall tar文件中提取源文件，然后在终端中更改为ConvertAll目录。对于基本安装，只需以root身份执行以下命令：python install.py

如果您的发行版默认为Python 2.x，您可能需要在这些命令中需要以python3代替python。

要查看所有的安装选项，使用：python install.py -h。要使用一个不同的预设（默认是 /usr/local）安装ConvertAll，使用：python install.py -p /prefix/path

Windows

要安装到所有用户，执行ConvertAll-x.x.x-install-all.exe文件。需要管理员权限。

要安装到单用户（不需要管理员权限），执行ConvertAll-x.x.x-install-user.exe文件。

对于便携式安装，执行ConvertAll-x.x.x-install-user.exe文件，取消选中快捷方式和卸载任务，选中便携式配置任务。

使用ConvertAll

基本

只需在“源单位”编辑窗口中键入一个单位名称。当您键入时，窗口下方的列表将会过滤仅显示匹配的单位。匹配的单位名称包含以键入的字词开头的字词。键入完整的单位名称/缩写或按返回键使用列表中高亮（蓝色字母）的单位。当然，用鼠标在列表中的单位上点击也会将单位添加到编辑窗口。您还可以使用上和下方向键从列表中高亮邻近的单位。

在“目标单位”编辑窗口中重复单位选择。当完成时，如果单位是兼容的，单位列表下方的数字编辑窗口会激活。可以在任一个数字窗口中输入数字，另一个窗口会显示转换结果。

组合单位

ConvertAll的真正实力在于其组合多个单位的能力。只需在单位名称之间键入‘*’或‘/’。允许输入像“ft*lbf”或"mi/hr"这样的单位。‘^’符号可以用于指数，例如“ft^3”或“ft*lbm/sec^2”。单位允许使用负指数，例如“sec^-1”（每秒），但可以切换到乘法或除法符号（“ft*sec^-2”变成“ft/sec^2”）。

乘法和除法具有相同的优先级，所以要从左到右对它们进行求值。括号也可用于对分母中的单位进行分组。所以“m/sec/kg”可以输入为“m/(sec*kg)”。含括号的版本可能不那么令人困惑。

单位文本框下方的按钮也会将运算符添加到在最靠近光标的激活单位。“平方”和“立方”按钮将添加或替换指数。“乘”和“除”按钮将会添加“*”和“/”运算符。

同样，从列表中点击一个单位通常会取代最靠近光标的单位。

运算符下方的“清除单位”按钮可以用于清空单位编辑窗口以便允许输入新的单位。

“过滤列表”按钮可以用于仅显示列表中的一种单位。注意，这不会显示可以组合形成类型的单位。

非线性转换

某些单位的转换是非线性的。这些例子包括华氏和摄氏温度尺度（由于偏移零点）和美国线规（对数）。非线性的单位已在注释列（位于“类型”列的右侧）中标明。

这些单位只有在它们没有和其他单位组合或与指数运算符一起使用时，才能转换。否则转换将没有意义。

快捷键

当输入单位名称时，空格会被忽略，因此可能会跳过它们。如果键入单位的复数形式，通常也会被忽略。 对于平方和立方单位（2或3的正指数），不需要键入“^”符号。

当部分输入的单位在列表中高亮时（蓝色字母），按enter键将会补全名称。上和下方向键可用于在按enter键之前高亮列表中邻近的单位。最靠近光标的单位将会被取代。

“最近的单位”按钮会打开一个最近使用的单位和单位组合的菜单。当前的单位组合将会被该菜单中的任何选择所取代。

tab键可以用于在“”源单位“和”目标单位测试框以及其数字编辑器之间循环。

要转换的数字可以在“源单位”或“目标单位”任意一方中输入。可以使用标准或科学计数，或者可以输入包含典型数学运算符（+、-、*、/、**）和括号的表达式。

选项

“选项...”按钮允许更改某些默认设置。这些设置将会自动保存，以便ConvertAll在重新启动时使用上次保存的设置。

第一个选项控制数值结果的显示，包括科学计数的使用和小数点位数。在将小数点位数设置为一个低的值时要小心，这可能会导致精确度的显著损失。推荐6个或以上位数。（默认为8个）

有一个用来设置要保存的最近的单位的数量的选项。设为零将禁用“最近的单位”按钮。另一个选项将会在启动时自动加载最近以前的单位。

有一个用来隐藏运算符文本选项按钮（第一行），和单位（第二行）的选项。如果键盘可以用于输入运算符，可以隐藏这些按钮来节省空间。

按钮对话框上还包含了控制文本框颜色的按钮。

命令行用法

转换可以在命令行（Linux或DOS控制台）中进行，无需调用图形界面。输入命令（Linux上是“convertall”，Windows二进制是“convertall_dos”）、数字、源单位和目标单位（用空格隔开）进行转换。包含空格的单位名称应该用引号括起来。或者，要提示每个单位条目，使用“-i”选项（Linux上是“convertall -i”，Windows是“convertall_dos -i”）。

转换完成之后，ConvertAll会提示输入新的数字进行相同的转换。或者可以输入“n”开始新的转换，“r”反向转换或“q”退出。

要获取更详细的选项列表，使用“-h”选项（Linux上是“convertall -h”，Windows上是“convertall_dos -h”）。
